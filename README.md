# Ride Day Grader

A stripped-down Next.js motorcycle ride-day grader designed to deploy cleanly on Vercel.

## Local run

1. Install Node.js 22.x
2. Run `npm install`
3. Run `npm run dev`
4. Open `http://localhost:3000`

## Deploy to Vercel

1. Upload this folder to GitHub
2. Import the repo into Vercel
3. Leave defaults alone
4. Ensure Node.js version is set to 22.x
5. Deploy

## Current UI changes included

- Ride windows and preferences are collapsed at the bottom
- Scoring model is collapsed at the bottom
- Combined ice / salt road-surface fail trigger
- Ice / salt risk visible in the Today card
- Café-racer motorcycle icon assets
